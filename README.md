A responsive Tourism Website built using HTML, CSS, Bootstrap, and JavaScript.
This project showcases popular tourist destinations with interactive sections, image carousels, and smooth navigation between pages.
Features
Home page with welcome section
Favourite tourist places list
Detailed view pages with image carousels
Smooth navigation between sections
Responsive layout using Bootstrap
Clean and simple user interface
